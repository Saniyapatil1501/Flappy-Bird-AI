# AI-Based Flappy Bird using Genetic Algorithm

This project implements a Flappy Bird game where birds learn automatically using a Genetic Algorithm.

## Features
- AI learns without manual coding
- Uses Neural Network + Genetic Algorithm
- Shows learning over generations
- Best bird highlighted visually

## Tech Stack
- Python
- Pygame
- NumPy
- Matplotlib

## How to Run
1. Install dependencies:
   pip install pygame numpy matplotlib

2. Run the game:
   python main.py
